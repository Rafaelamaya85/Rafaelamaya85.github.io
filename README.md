# Personal Introduction Website

## Overview

This project is a personal introduction website created for Project 02 in CMPA 3301 at Texas Tech University. It showcases my ability to build a structured webpage using HTML and CSS while applying professional development practices such as version control with Git and website deployment through GitHub Pages.

As a senior pursuing a Bachelor of Science in Computing Applications at Texas Tech University, I created this website to introduce myself, share my interests in technology, and talk about my education and career goals. This project is another step in building my professional portfolio to help me grow my career in technology. It also gives me a place to show my skills and the work I've completed as I prepare to apply to graduate school and pursue a master's degree.

## Features

- Semantic HTML page structure
- CSS stylesheet for consistent design
- Flexible layout that works across desktop and mobile devices
- About Me section
- Skills and interests section
- Contact information
- Public deployment using GitHub Pages

## Technologies Used

- HTML
- CSS
- Git
- GitHub
- GitHub Desktop
- GitHub Pages
