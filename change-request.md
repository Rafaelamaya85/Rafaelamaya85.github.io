# Change Request

## Proposed Change

A key stakeholder has requested adding an embedded Google Map of my hometown, Irving, Texas, to my personal project website. The map would be placed on the page so visitors can easily see where I am from.

## Reasons for the Change

The purpose of this change is to make the website more engaging and provide a deeper understanding of who I am. Since the website is an introduction, adding a map gives visitors a window into my hometown and my roots. They will see how close I grew up to Texas Stadium and understand my love for the Cowboys. It also adds an interactive feature to the site helping it standout.

## How This Change Will Add Value

This change will give visitors a window to see into my past. To visualize Irving, Texas as more than just words on a page. Just like the San Fernando Valley was an important part of Smalls' story in The Sandlot, Irving is an important part of mine. Adding a map gives visitors a visual starting point to understand where I grew up and the community that helped shape me.  It also makes the website look a little more polished without changing its original purpose.

## How This Change Will Affect the Constraints

- **Time:** This change would require a few hours of additional research and testing time to find the Google Maps embed code, work with the design of the webpage, and test that it works correctly.

- **Quality:** The quality of the website would improve because it adds a visual feature that breaks the monotony of just plain text on a page. I would need to work on the page layout to make sure the map fits, makes sense and works properly on both mobile and desktop screen sizes.

- **Budget:** There would be no additional cost or change to the budget of the project because Google Maps allows for the code to be embedded for free. No additional software or paid services would be required.
